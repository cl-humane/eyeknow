1. Download Db Browser:
https://sqlitebrowser.org/dl/

2. Open Visual Studio Code.

3. In the terminal of Visual Studio Code, run:
	pip install flask
	pip install bcrypt

4. Go to add_table.py file
	Run this file in the terminal:
		python add_table.py

5. Go to file named add_admin.py.
	Run this file in the terminal:
		python add_admin.py

6. Go to file named login.py.
	Run this in the terminal:
		python login.py

You will see this:
http://127.0.0.1:5000/
